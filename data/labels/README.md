# data/labels/ — ML 타깃(라벨): 침수흔적도 · NDMS

머신러닝 학습의 양성(침수=1) 라벨 자료를 넣는 공간. **원시 또는 이미 격자화된 형태 모두 가능**.

**자료:**
- **침수흔적도 / 침수위험지도** (폴리곤) — 침수 범위.
- **NDMS(국가재난관리시스템) 피해자료** (포인트) — 개별 피해 발생.

**형식:** 원시(폴리곤/포인트 SHP·GPKG) 또는 사전 격자화(100m GeoTIFF).
**변환 코드:** `src/standardize/labels_build.py` (원시 → 격자화, 두 출처 합집합; 격자 제공본 → `verify_grid.py`로 검증 후 사용).
**담당:** geo-data-engineer(라벨 생성) → flood-risk-modeler(학습).

> 양성(1) = 침수흔적도 ∪ NDMS. 각 셀에 출처(폴리곤/포인트)·연도·사상을 태그로 남긴다. 침수 셀은 극소수 → 불균형·음성 표본 설계는 modeler와 협의. "미기록 = 미침수"가 아님에 유의(NDMS 도시 신고 편향 포함).
