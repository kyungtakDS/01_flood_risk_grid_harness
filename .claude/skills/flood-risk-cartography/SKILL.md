# Flood Risk Cartography — 위험지도·산출물 제작

모델·시나리오가 만든 위험지수 격자를 의사결정자가 읽을 수 있는 지도·통계·리포트로 변환한다.

## 입력
- `_workspace/04_risk_index.tif`(현재 위험), `04_forecast_{RP}yr.tif`(시나리오), `04_uncertainty.tif`.
- 모두 마스터 격자 정렬 전제. 산출 전 계약 정합을 재확인(어긋난 입력은 지도도 어긋난다).

## 위험등급 분류
연속 위험지수(0~1)를 등급(예: 매우낮음~매우높음 5단계)으로 나눈다.
- **Jenks 자연분류**: 데이터 자연 군집 반영, 일반 위험지도에 적합.
- **분위(quantile)**: 등급별 면적 균등, 지역 비교에 유리.
- **고정 임계**: 방재 기준·재현주기 정책값이 있으면 사용(지도 간 비교 일관성).
- 분류 방식·경계값을 범례·메타에 명시. 시나리오 간 비교 지도는 **동일 분류 기준**을 써야 변화가 왜곡되지 않는다.

> 색상 설계는 순차형 팔레트(저→고)를 쓰고, 색각 이상을 고려한다. 위험 상승은 직관적으로 진한 색으로. 불확실성이 큰 셀은 별도 레이어/해칭으로 표시해 과신을 막는다.

## 배포 포맷
- **COG GeoTIFF**: 등급/지수 래스터. 내부 타일 + 오버뷰로 웹에서 부분 로드.
- **웹타일**: 필요 시 XYZ/MBTiles로 타일링.
- **벡터 요약**: 행정구역·유역 단위 위험 통계 GeoParquet/GeoJSON.

## 통계·리포트
- 위험등급별 면적, 등급별 노출인구·건물·자산(노출 인자와 교차), 재현주기별 위험 증가.
- 고위험 상위 지역 목록(방재 우선순위). 시나리오 간 신규 위험 진입 지역.
- 리포트 구조:
  ```
  # 전국 홍수위험도 평가 리포트
  ## 요약 (핵심 수치)
  ## 방법 (자료·모델·재현주기)
  ## 위험 분포 (등급별 면적·인구)
  ## 시나리오 비교 (재현주기별)
  ## 고위험 지역 및 방재 시사점
  ## 한계·불확실성 (라벨·모델·격자)
  ```

## 출력
- `_workspace/06_riskmap_current.tif`, `06_riskmap_{RP}yr.tif` (등급화 COG)
- `_workspace/06_risk_stats.{csv,geojson}` — 지역·유역별 통계
- 최종 리포트: 사용자 지정 경로 또는 `06_flood_risk_report.md`

## 팀 협업 요점
- `scenario-forecaster`로부터 시나리오 목록 수신 → 동일 분류 기준으로 지도화.
- `exposure-vulnerability-analyst`의 노출 인자와 교차해 노출인구·자산 통계 산출.
- `validation-inspector`의 불확실성·한계를 리포트에 반영(과신 방지).

## 이전 산출물이 있을 때 (재호출)
`06_*`가 존재하면 변경된 위험지수/시나리오만 재지도화. 분류 기준을 바꾸면 비교 대상 지도 전체를 동일 기준으로 재생성한다.
