# 수자원단위지도 기반 유역 분할 & 격자 파티셔닝

전국 홍수위험도는 **수자원단위지도**(환경부/K-water)의 유역 위계를 골격으로 격자망을 구성하고, 처리·모델링을 유역 단위로 분할한다. 이 문서는 그 규약을 정의한다.

## 목차
1. 유역 위계
2. 격자 파티셔닝 규칙
3. 흐름기반 인자의 완전유역 처리
4. 중권역 단위 ML 모델링

---

## 1. 유역 위계 (수자원단위지도)

| 위계 | 개수(대략) | 코드 자릿수 | 용도 |
|------|-----------|------------|------|
| **대권역** | 5(+제주 등) | 1자리 계열 | 격자망 처리 단위(전국 분할). 한강·낙동강·금강·영산강·섬진강 |
| **중권역** | ~117 | 상위 코드 | **ML 모델링 단위**(부하 저감). 공간 CV 블록 |
| 표준유역 | ~850 | 세부 코드 | 세부 수문 집계·검증 단위 |

- **대권역**: 한강권역, 낙동강권역, 금강권역, 영산강권역, 섬진강권역 (제주·기타 도서는 별도 처리).
- 각 유역은 유역코드(hydro unit code)를 키로 가지며, 격자 속성테이블에 `daegwon`, `junggwon`, `std_watershed` 컬럼으로 부착한다.

## 2. 격자 파티셔닝 규칙

전국을 한 번에 처리하면 메모리·연산 부하가 크므로 유역 단위로 분할하되, **모든 타일은 국가 표준 격자 원점에 snap된 좌표를 공유**해야 한다.

- 마스터 격자는 대권역 단위로 생성(`grid_master_{daegwon}.tif`)하거나, 전국 마스터를 중권역 타일로 잘라 사용.
- **경계 정합 불변식**: 인접 유역 타일의 격자가 정확히 맞물려야 한다. 각 타일 원점 = 국가 원점 + (정수 × 100m). 임의 bbox 원점을 쓰면 유역 경계에서 반셀(50m)씩 어긋나 전국 모자이크가 깨진다.
- 유역 경계에 걸친 셀은 우세 유역으로 귀속(중복 계산 금지). 귀속 규칙을 태그에 기록.

> **왜 국가 원점 snap인가:** 유역별로 격자를 독립 생성하면 각자 다른 원점을 가져 경계에서 어긋난다. 국가 원점에 snap하면 어느 유역에서 만든 셀이든 동일 격자에 정렬되어, 나중에 전국 위험지도로 이음매 없이 합쳐진다. 이것이 §grid-standard.md 불변식(원점·transform 일치)의 유역판이다.

## 3. 흐름기반 인자의 완전유역 처리 (중요)

경사·거리 같은 국지 인자는 타일 단위로 계산해도 되지만, **흐름누적·하천망·HAND·유역경계 같은 흐름기반 인자는 완전한 유역 위에서 계산해야 한다.**

- 흐름누적을 중권역 타일 경계에서 자르면 상류 집수가 잘려 물길이 조각난다(치명적 오류).
- 따라서 흐름기반 인자는 **대권역(완전 유역) 단위로 계산**하거나, 최소한 상류를 포함하는 버퍼를 두고 계산한 뒤 중권역으로 잘라 쓴다.
- 처리 순서: DEM 함몰처리 → 대권역 흐름누적/하천망/HAND 산출 → 중권역으로 타일링 → 모델링.

## 4. 중권역 단위 ML 모델링

ML 모델링(RF·XGBoost·LightGBM·CatBoost·DecisionTree)은 **중권역 단위로 수행**한다.

**이유와 이점:**
- **부하 저감**: 전국 수백만 셀을 한 번에 학습하지 않고 중권역 블록으로 나눠 메모리·연산을 관리.
- **수문 일관성**: 중권역은 수문학적으로 응집된 단위라, 인자-침수 관계가 지역 내에서 일관.
- **공간 CV 자연 정합**: 중권역 = 공간 블록 CV의 fold. 인접 셀 누수를 막아 성능을 정직하게 평가(무작위 k-fold의 공간 누수 방지).

**모델링 전략 선택 (라벨 밀도에 따라):**
| 전략 | 언제 | 주의 |
|------|------|------|
| 중권역별 개별 모델 | 해당 중권역에 침수 라벨(침수흔적도·NDMS)이 충분할 때 | 라벨 희소 중권역은 과적합/학습 불가 |
| 대권역 공유 모델 + 중권역 예측 | 중권역 라벨이 희소할 때 | 대권역 내 라벨 풀링, 중권역 타일로 예측 |
| 전국 모델 + 중권역 타일 처리 | 라벨이 전반적으로 희소할 때 | 부하는 타일 처리로 저감, 지역 특성은 유역 인자로 반영 |

**권장:** 라벨 밀도를 먼저 진단하고, 중권역 라벨이 충분하면 개별 모델, 희소하면 대권역 풀링으로 상향 조정한다. 어떤 전략이든 **중권역을 CV fold·처리 단위로 사용**하는 원칙은 유지한다. 선택 근거를 모델 카드에 기록.

**산출 통합:** 중권역별 예측 위험지수를 국가 원점 정렬 덕분에 이음매 없이 모자이크 → 전국 위험지도. 유역 경계 셀 이중계산이 없는지 검증자가 확인.
