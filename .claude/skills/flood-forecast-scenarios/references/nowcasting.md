# 실시간 Nowcasting 확장

운영·경보 목적으로 수시간 내 홍수위험을 실시간 예측할 때만 로드한다. 배치성 재현주기 예측과 달리 실시간 스트림을 다룬다.

## 원리
학습된 위험 모델의 강우 인자를 **실시간 관측·초단기예보 강우**로 교체해 현재~수시간 후 위험을 반복 산출한다. 지형·노출·취약성은 고정, 강우만 실시간 갱신된다.

## 절차
1. 실시간 강우 입력: 기상청 초단기예보(동네예보 API), 레이더 합성강우(HSR), AWS 실황.
2. 선행강우 반영: 최근 누적강우(선행토양수분)를 인자로 추가하면 유출 반응을 보정.
3. 마스터 격자로 즉시 정합(재투영·리샘플) → 강우 인자 갱신.
4. 모델 예측 → 실시간 위험 래스터. 임계 초과 셀에 경보 플래그.
5. 갱신 주기(예: 10분/1시간)로 반복.

## 운영 요건
- 지연(latency) 최소화: 무거운 지형 인자는 사전 계산해 캐시, 강우만 갱신.
- 격자 정합은 실시간에도 필수 — 어긋난 강우 격자는 잘못된 경보를 낸다.
- 임계값은 재현주기 예측·과거 사상으로 사전 보정.

## 출력
- `04_forecast_nowcast_{timestamp}.tif` (시각 태그)
- 경보 대상 격자 목록(임계 초과 셀 + 노출인구).
